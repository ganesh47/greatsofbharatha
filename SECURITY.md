# Security Policy

## Supported Versions

This project is pre-1.0 and currently supports security fixes on the `main` branch only.

## Reporting a Vulnerability

Please do not disclose security issues publicly first.

Report suspected vulnerabilities through one of these paths:
- GitHub private vulnerability reporting for this repository, if enabled
- direct contact with the repository owner

Please include:
- a clear description of the issue
- affected files, workflows, or components
- reproduction steps or proof of concept
- potential impact
- any suggested mitigation

We will aim to:
- acknowledge receipt promptly
- assess severity and exposure
- remediate in a reasonable timeframe
- coordinate disclosure once a fix is available

## Scope Notes

This repository includes:
- iOS app source code
- GitHub Actions workflows
- release/TestFlight automation
- CI helper scripts

Security reports involving secrets exposure, workflow abuse, supply-chain risk, or App Store Connect automation are in scope.
